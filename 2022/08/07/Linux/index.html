Linux设置环境变量

设置完成使用该命令刷新

source 编辑的文件

临时设置

export PATH=/you/bin/path:$PATH

当前用户永久生效

vim ~/.bashrc
export PATH=/you/bin/path:$PATH

所有用户永久生效（需要重新登录）

//三者挑其一
vim /etc/bashrc
vim /etc/profile
vim /etc/environment
//添加下面的设置
export PATH=/you/bin/path:$PATH

问题：安全检查

To add an exception for this directory, call:
git config --global --add safe.directory /media/data/users/jhu3szh/serialize

//修改拥有者

//无效再使用一下方式

问题：Snap商店安装错误

问题描述：

snap “xxxx” has “install-snap” change in progress

解决方法：

//查看snap 正在安装的软件，获取id
snap changes

//终止安装软件
sudo snap abort id

//重新安装软件即可
sudo snap install xxx

Cat日志查看

cat常用于查看指定日志信息，系统在运行时，日志会不断写入，如果使用vim等编辑器指令查看会无法得到全部的日志，同时也可能导致日志的修改。

//查看文件的全部内容
cat 文件名 
//如：cat test.js

//对所有行添加编号输出，1开始
cat -n 文件名
//如：cat -n test.js

//排除空白行添加编号输出
cat -b 文件名
//如：cat -b test.js

//替换连续两行以上的空白行为一行空白行
cat -s 文件名
//如：cat -b test.js

//结合grep查看指定内容
cat 文件名 | grep 搜索内容

Grep查看符合条件的内容

grep 查找内容 文件名
//如：grep hello test.js

//忽略大小写
grep -i 查找内容 文件名

Find查找文件位置

find 路径 <选项> 参数

//查找当前目录及子目录test.js
find . -name test.js

//查找所有目录test.js
find / -name test.js
