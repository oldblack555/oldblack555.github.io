【问题】：使用Lombok找到get、set等问题（Idea）

解决方法

  1. 打开setting的plugins，找到lombok插件并qiyong
  2. 找到setting的Java Compiler选择Use compiler为Javac
  3. 找到setting的Annotation Processors选中Enable annotation processing、Obtain processors from project classpath、Module output directory
  4. （针对Gradle）找到setting的Build Tools的Gradle，将Build and run using和Run tests using改为IntelliJ IDEA
