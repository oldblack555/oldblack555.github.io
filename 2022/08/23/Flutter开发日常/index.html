Flutter开发Linux程序中文乱码解决

原因

程序找不到字体

解决方法

  • 项目根目录下新建fonts文件夹
  • 将字体文件导入fonts文件夹下（字体来源：网上下载，Windows的Fonts文件夹下拷贝，linux的/usr/share/fonts文件夹拷贝）
  • 在pubspec.yaml下配置fonts
flutter:
  uses-material-design: true
  #添加font文件
  fonts:
    - family: cn
      fonts:
        - asset: fonts/msyh.ttc
  • 在MaterialApp中声明fontFamily
MaterialApp(
      title: '重命名神器',
      theme: ThemeData(
        primarySwatch: Colors.blue,
        //声明font
        fontFamily: 'cn',
      ),
      home: const HomeUI(),
);

Flutter开发Web中网络图片跨域问题
