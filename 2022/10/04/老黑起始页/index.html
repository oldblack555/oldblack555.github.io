老黑起始页

介绍

老黑起始页是一个仿终端的web起始页，模拟终端命令，通过简单的命令实现一般起始页的功能。如常规的搜索功能，使用search命令加搜索内容便可快速进行搜索。项目处于属于个人喜好，偏自娱自乐，可能存在许多不足之处，还请大佬们指导哈。

指令集

辅助参数：–help，负责提示指令参数及作用，或者收藏的标签页等。

系统指令

  • info

    显示所有的指令及作用

  • bg、background

    设置背景图（目前只支持网络图片）

  • clear

    清除消息列表

  • config

    系统属性设置指令，如背景色、透明度、字体大小等。参数及说明如下：

    参数 解释
    -bg 设置背景颜色，可选颜色[red、black、blue、green、pink、purple、white、orange、yellow]。
    -ba 设置背景透明度，取值范围[0-1]。
    -se 设置默认的搜索引擎。可使用引擎[chrome、bing、baidu]。
    -fs 设置字体大小，最小字体为10，默认为20。
    -fc 设置字体颜色，可选颜色同bg参数
    -blur 设置背景模糊效果，非负数，默认为10
    -blog 指定个人博客地址，默认为作者的博客

用户指令

  • love

    标签管理

    参数 说明
    -a 添加新的标签
    -r 重命名标签
    -d 删除标签

  • open、cd

    打开网址或者收藏的标签

  • search

    搜索指定内容

    参数 说明
    -e 指定搜索引擎（chrome、bing、baidu）

直接指令

  • bili、bilibili

    打开B站首页

  • blog

    打开个人博客，可通过config指令指定自己的博客地址

  • github

    打开GitHub主页

  • stackof

    打开StackOverflow主页

娱乐指令

  • music

    随机播放音乐，开发中，可使用，但未完善。

    参数 说明
    -stop 停止播放
    -start 开始播放
